# Lumen Core API Examples

Complete code examples for integrating with Lumen Core API.

## Quick Start

```bash
cd examples
npm install
npm start
```

## Run Individual Examples

```bash
npm run example1   # Simple chat
npm run example2   # Terminal command
npm run example3   # Multi-turn conversation
npm run example4   # Code generation
npm run example5   # War Room code review
npm run example6   # WebSocket streaming
npm run example7   # Session management
npm run example8   # Error handling
npm run example9   # Batch processing
npm run example10  # Resilient WebSocket
```

## Examples Overview

| Example | Description | Features |
|---------|-------------|----------|
| 1 | Simple Chat | Basic API usage |
| 2 | Terminal Command | Command execution |
| 3 | Conversation | Memory persistence |
| 4 | Code Generation | AI code writing |
| 5 | War Room | Multi-agent review |
| 6 | WebSocket | Real-time streaming |
| 7 | Session Management | Memory control |
| 8 | Error Handling | Error patterns |
| 9 | Batch Processing | Parallel requests |
| 10 | Resilient WebSocket | Auto-reconnect |
